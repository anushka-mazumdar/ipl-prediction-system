# IPL Prediction System 🏏

This project predicts IPL match outcomes, team scores, and player performance metrics using machine learning and large language models (LLMs). It is built with Django, FastAPI, and supports live match updates and chatbot explanations.

## 📁 Project Structure

- `ml_model/`: Machine learning models and training code
- `data_processing/`: Feature engineering and preprocessing
- `backend_fastapi/`: FastAPI backend to serve predictions
- `backend_django/`: Django REST backend with DB integration
- `llm/`: Local LLM integration for reasoning/explanations
- `chatbot/`: Interactive chatbot (UI/backend)
- `notebooks/`: EDA and experiments
- `tests/`: Unit and integration tests
- `docs/`: Swagger/OpenAPI, architecture diagrams
